Space Weather Model Staging Platform

Greg's current programming work involves creating a Model Staging Platform on public cloud architectures to lower the barrier to entry for running research code and getting the cutting edge research into operational centers sooner. An example of this is developing a web interface to an empirical Fortran model so that anyone can interact with the model through a browser without having to worry about compilers or how to run the code. The project involved creating a Python wrapper around the Fortran code (pymsis) and setting up a serverless application (msis-api) to call the Fortran code with a web API. The end product is the MSIS Visualizer website that was created with Javascript's d3 library to plot surface geographic fields and altitude profiles of atmospheric constituents depending on the space weather drivers.

Some other projects that have followed a similar architecture are: VECTOR, a satellite drag calculator that enables users to upload a satellite design and choose the orientation and atmospheric conditions to provide a detailed assessment of drag on the vehicle; enlilviz, a solar wind visualization program in Python to produce animations and interactive plots for Space Weather Forecasters to use in their operational environments.

Space Weather Research

In addition to developing tools for others in the community, Greg also continues to advance the state of the art research in applied machine learning techniques to the spatial and temporal interpolation of magnetic fields. For these techniques he wrote a Spherical Elementary Current System code (pysecs) to reduce the problem to a set of divergence-free basis functions, which ensures the underlying physical constraints and properties are maintained while interpolating. He and his colleagues then trained a deep neural network to fit these basis functions using synthetic observations from expensive HPC models that can't be run in real-time. This technique of reducing the dimensionality with physical basis functions and training with large-scale HPC models has shown promise for creating more realistic magnetic field interpolations across North America (poster link).
